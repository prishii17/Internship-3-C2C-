-What is HTML?
    Hyper Text Markup Language
    not a programming language
    it is a markup language used to create the structure and content of a webpage
 
 -Hyper Text : refers the 'links' that connects web pages
 -Markup Language : refers to the 'tags' like (<h1> or <p>)
