What is CSS?

CSS (Cascading Style Sheets) is used to style and design web pages. It controls the appearance of HTML elements such as colors, fonts, spacing, layouts, animations, and responsiveness.

Why CSS is Used
Improves website appearance
Separates design from content
Makes websites responsive
Enhances user experience
Allows reuse of styles across multiple pages
Types of CSS
1. Inline CSS

CSS is written directly inside an HTML element using the style attribute.

<p style="color:red;">Hello World</p>
Advantages
Quick styling for a single element
Priority

Inline CSS has the highest priority among normal CSS rules.

2. Internal CSS

CSS is written inside the <style> tag within the HTML document.

<head>
    <style>
        p{
            color:blue;
        }
    </style>
</head>
Advantages
Useful for styling a single page
3. External CSS

CSS is written in a separate .css file and linked to the HTML document.

<link rel="stylesheet" href="style.css">
p{
    color:green;
}
Advantages
Reusable
Easy maintenance
Better organization
CSS Priority Order
Inline CSS
Internal CSS
External CSS
!important

The !important keyword can override normal priority rules.

p{
    color:red !important;
}
CSS Selectors

Selectors are used to target HTML elements for styling.

1. Universal Selector (*)

Selects all elements.

*{
    margin:0;
    padding:0;
}
2. Child Selector (>)

Selects direct child elements.

div > p{
    color:red;
}
3. Group Selector (,)

Applies the same style to multiple elements.

h1, h2, p{
    color:blue;
}
4. Descendant Selector (space)

Selects elements inside another element.

div p{
    color:green;
}
5. Adjacent Selector (+)

Selects the immediate next sibling.

h1 + p{
    color:orange;
}
6. Pseudo Class (:)

Defines a special state of an element.

button:hover{
    background-color:yellow;
}
Common Examples
:hover
:focus
:first-child
:last-child
7. Pseudo Element (::)

Styles a specific part of an element.

p::first-letter{
    font-size:30px;
}
Common Examples
::before
::after
::first-letter
::first-line

7. Responsive Design (Media Query)

Used to make websites adapt to different screen sizes.

@media(max-width:768px){
    body{
        background-color:lightblue;
    }
}
8. Animation

Adds movement and visual effects.

    from{
        left:0;
    }
    to{
        left:100px;
    }
}

    animation:move 2s infinite;
}
